# Data Redundancy Removal System

A cloud-based application that prevents duplicate records using MongoDB Atlas and Flask.

## Features

- Add Records
- Duplicate Detection
- Email Validation
- Duplicate Email Prevention
- MongoDB Atlas Integration
- Docker Containerization
- GitHub Actions CI/CD
- Cloud Ready Deployment

## Technologies Used

- Python
- Flask
- MongoDB Atlas
- Docker
- GitHub Actions

## Run Locally

```bash
pip install -r requirements.txt
python app.py
